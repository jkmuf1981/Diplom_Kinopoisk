# Diplom_Kinopoisk

Описание проекта
Проект представляет собой финальную работу по ручному тестированию сервиса Кинопоиск. 
Цель проекта — проверка функциональности сайта и приложения Kinopoisk с использованием автоматических инструментов 
тестирования.

Продукт
Кинопоиск — крупнейший русскоязычный сервис о киноиндустрии и кинопрокате. 
Сайт предоставляет подробную информацию о фильмах, сериалах, персонах индустрии, 
а также позволяет пользователям оценивать контент, оставлять отзывы и приобретать билеты в кинотеатр.

Основные возможности сайта включают:

Поиск фильмов и сериалов,
Просмотр кадров, трейлеров, постеров и обоев, добавление фильмов и сериалов в избранное.
Онлайн-просмотр фильмов и сериалов через платную подписку («Яндекс.Плюс») или отдельно.
Приложение доступно на разных устройствах:
Смартфоны (Android/iOS),
Умные телевизоры (LG/Samsung/Android TV),
Игровые приставки (PlayStation).

Инструкция по работе с тестами
Подготовка среды:
Склонируйте репозиторий проекта командой:git clone <репозиторий_проекта>
Установите необходимые библиотеки командой:pip install -r requirements.txt
Откройте файл config.py, добавьте ваш личный ключ для доступа к API. ??????

Запуск тестов:
1. Тестирование API
Запуск тестов API:pytest tests/test_api.py --alluredir=./allure_result_api

Затем создайте отчёт Allure:allure serve allure_result_api

2. Тестирование UI
Для запуска UI-тестов выполните:pytest tests/test_ui.py --alluredir=./allure_result_ui

Отчёт создаётся аналогично:allure serve allure_result_ui

3. Общий запуск всех тестов
Запустить все тесты:pytest --alluredir=./allure_result_all

Создать общий отчёт:allure serve allure_result_all

Используемые технологии
Pytest: Основная библиотека для написания и выполнения тестов.
Selenium: Библиотека для автоматизации действий в браузере (UI-тестирование).
Requests: Используется для отправки запросов к API (API-тестирование).
Allure: Генератор отчётов для визуализации результатов тестирования.
Таким образом, проект обеспечивает комплексный подход к проверке работоспособности сервиса Кинопоиск, 
используя современные инструменты автоматического тестирования.
